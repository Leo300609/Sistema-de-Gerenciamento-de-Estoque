## 🚀 Sobre o Projeto

O **SGE** é uma solução robusta focada na eficiência operacional de empresas que necessitam de um controle rigoroso de entrada e saída de materiais. O software visa reduzir perdas e otimizar o fluxo de reposição.

### 🛠 Tecnologias e Ferramentas
* **Linguagem:** JavaScript / Node.js
* **Lógica:** Estruturas de repetição e manipulação de arrays (CRUD)
* **Planejamento:** Pesquisa de campo e Wireframing
* **Versionamento:** Git & GitHub Actions (CI/CD)

## 🏗️ Estrutura da Lógica (V0.1)

Atualmente, o sistema conta com um motor de lógica em `logica-estoque.js` que gerencia:
- [x] **Create:** Cadastro de novos produtos.
- [x] **Read:** Consulta de itens no array.
- [x] **Update:** Reposição automática de estoque.
- [x] **Security:** Verificação de saldo disponível para saídas.
